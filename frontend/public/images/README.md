# Images Setup Guide for Herizon Landing Page

This directory contains the images used in the Herizon landing page. To make your website look professional and user-friendly, you need to add 3 images here.

## Required Images

### 1. Hero Image (`hero.png`)
- **Location:** `/images/hero.png`
- **Recommended Size:** 800x600px or 1200x900px
- **Description:** Main hero section image showing a professional woman balancing work and life
- **Suggestions:**
  - A happy single mother working on laptop with child nearby
  - Professional woman in a modern workspace
  - Woman using smartphone with confident smile
  - Team collaboration or support group image

### 2. Product Dashboard (`product.png`)
- **Location:** `/images/product.png`
- **Recommended Size:** 800x600px or 1200x900px
- **Description:** Screenshot or mockup of the Herizon dashboard interface
- **Suggestions:**
  - Dashboard mockup showing calendar, tasks, and wellbeing metrics
  - Mobile app interface preview
  - Desktop application screenshot
  - UI/UX design showing the platform's features

### 3. Community Impact (`community.png`)
- **Location:** `/images/community.png`
- **Recommended Size:** 800x600px or 1200x900px
- **Description:** Image representing community support and connection
- **Suggestions:**
  - Group of diverse women supporting each other
  - Women networking or in a support group
  - Community meeting or mentorship scene
  - Mothers helping each other with childcare

## How to Add Images

1. Download or create your 3 images
2. Rename them to:
   - `hero.png`
   - `product.png`
   - `community.png`
3. Place them in this directory: `frontend/public/images/`
4. The website will automatically load them

## Fallback Images

The landing page is configured with fallback images from Unsplash if your images are not found. However, for the best user experience, add your own images that match your brand and message.

## Image Optimization Tips

- Use PNG format for best quality
- Compress images to reduce load time (use tools like TinyPNG)
- Maintain aspect ratio around 4:3 or 16:9
- Ensure images are high-resolution for retina displays
- Use consistent color tones across all images
