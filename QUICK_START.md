## Get Your Website Running (3 Easy Steps)

### Step 1: Start the Development Server
```bash
cd frontend
npm run dev
```

Your website will open at: **http://localhost:5173** (or 5174 if 5173 is busy)

### Step 2: Add Your 3 Images (Optional but Recommended)

Navigate to: `frontend/public/images/`

Add these 3 images:
1. **hero.png** - Main hero image (woman balancing work/life)
2. **product.png** - Dashboard or app interface screenshot
3. **community.png** - Community support or group image

**Recommended Size:** 800x600px or larger (PNG format)

**Don't have images?** No problem! The page uses professional Unsplash images as fallbacks.

### Step 3: View Your Professional Landing Page

Open browser → **http://localhost:5173**

That's it! 🎉

## Need Help?

### Common Issues:

**Problem:** Images not showing
**Solution:** Make sure images are named exactly: `hero.png`, `product.png`, `community.png` (lowercase)

**Problem:** Port already in use
**Solution:** Vite will automatically use the next available port (5174, 5175, etc.)

**Problem:** Styles not updating
**Solution:** Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
